# ⚡ CYBER CONVERT

[![Banner](https://img.shields.io/badge/AESTHETIC-CYBERPUNK-FF00FF?style=for-the-badge&logoColor=white)]()
[![Tech](https://img.shields.io/badge/TECH-VANILLA_JS-00FFFF?style=for-the-badge&logoColor=black)]()

**CYBER CONVERT** is a high-performance, neon-infused unit conversion tool designed with a futuristic "Cyberpunk" aesthetic. It provides instant, precise conversions between common units of length (CM, Feet, Miles) through a sleek, interactive interface.

---

## ✨ Features

- 🌌 **Neon Aesthetic**: A premium "Cyberpunk" design featuring vibrant glows and high-contrast visuals.
- ⚡ **Instant Initialization**: Quick calculation engine with specialized modules for each unit.
- 📱 **Responsive Interface**: Built for clarity and speed, focusing on specific unit outputs.
- 🛠️ **Modular Logic**: Cleanly separated conversion logic using ES Modules for maintainability.

---

## 🚀 Tech Stack

- **Core**: HTML5, Vanilla CSS3
- **Scripting**: Modern JavaScript (ES Modules)
- **Typography**: [Orbitron](https://fonts.google.com/specimen/Orbitron) (Futuristic Sans-Serif)

---

## 📂 Project Structure

```text
Unit_And_Dimenstional/
├── index.html       # Main application entry point
├── style.css        # Neon-infused styling definitions
├── index.js         # Core application Controller
├── cal.js           # Calculation Orchestrator
├── cm.js            # CM Conversion Module
├── feet.js          # Feet Conversion Module
└── miles.js         # Miles Conversion Module
```

---

## 🛠️ Usage

1. **Enter Value**: Input the numeric value you wish to convert.
2. **Select Unit**: Choose your source unit from the dropdown.
3. **Initialize**: Click the **INITIALIZE** button to see the conversion details.
4. **Results**: View the converted values in the dedicated result tiles.

---

## ⚙️ Installation

To run this project locally, simply clone the repository and open `index.html` in your browser. No build steps are required as it uses native ES Modules.

```bash
# Clone the repository
git clone https://github.com/[username]/Cyber-Convert.git

# Navigate to the directory
cd Unit_And_Dimenstional

# Open index.html
# (Tip: Use a local server like "Live Server" for the best experience with ES Modules)
```
